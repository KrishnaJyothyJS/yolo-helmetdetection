
# 👷‍♂️ AI-Based Helmet Detection System

An efficient, real-time computer vision system built with YOLOv8 and OpenCV to detect safety compliance. This project was developed for the AI Helmet Detection Competition and is capable of processing live webcam feeds and pre-recorded media to identify individuals with and without helmets.

---

## 🚀 Key Features

* **Real-Time Live Inference:** Processes live webcam feeds with an integrated FPS counter to ensure smooth performance.
* **Media Testing Pipeline:** Includes a dedicated script to test static images and pre-recorded videos, automatically saving the annotated results.
* **Custom 3-Class Detection:** Specifically trained to recognize the context of the scene with three distinct classes:
    * 🟩 **Helmet** (Safety Compliant)
    * 🟥 **No-Helmet** (Safety Violation)
    * 🟦 **person** (Contextual Background)
* **Visual Alert System:** Dynamically color-coded bounding boxes for instant visual feedback on safety violations.
* **Hardware Optimized:** Codebase is structured to easily switch between CPU inference (for development) and GPU training/inference (CUDA) for high-speed performance.

**💻 Usage Instructions**

1. Live Webcam Demo (Real-Time Detection)
To run the real-time detection system using your connected webcam:
python src/detect_live.py
  (Note: Press q while the video window is active to exit the stream.)

2. Testing Images & Videos (Offline Media)
To test the model against specific edge cases, images, or street footage:
python src/test_media.py
  (The script will prompt you for the file path (e.g., test_assets/video.mp4). Once finished, it will automatically open the media_test folder to show you the annotated results with custom bounding boxes.)

Training the Model
To train the model:
python src/train.py


**🧠 Model Details**
Our system uses YOLOv8, one of the world's fastest and most accurate AI models for "seeing" objects in real-time.

Custom Training: We didn't just use a generic model. We specifically trained this AI to distinguish between people wearing helmets and those who aren't.

Three-Way Recognition: The model is smart enough to identify three distinct things: Helmets, No-Helmets, and the Person involved.

Real-Time Speed: It is optimized to process video instantly. This means it can detect safety violations on a live camera feed without any lag.
